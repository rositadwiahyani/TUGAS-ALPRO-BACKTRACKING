# Deskripsi Program #
Program ini merupakan implementasi algoritma Backtracking untuk menyelesaikan masalah Rat in a Maze. Seekor tikus harus mencari jalan keluar dari labirin dengan mencoba berbagai kemungkinan jalur. Jika jalur buntu, program akan kembali ke posisi sebelumnya (backtracking) untuk mencoba jalur lain sampai menemukan jalan keluar.

# Algoritma yang Digunakan #
Backtracking

# Tujuan Program #
1. Menemukan jalur dari titik awal ke titik akhir labirin
2. Memvisualisasikan proses pencarian jalur
3. Menunjukkan proses backtracking saat jalur buntu

# Fitur Program #
1. Visualisasi labirin di terminal  
2. Menampilkan pergerakan tikus secara bertahap  
3. Menampilkan proses backtracking  
4. Titik Start dan End jelas  
5. Tampilan labirin rapi dengan frame
